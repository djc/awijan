# Awijan: create release notes from PR history

```
Usage: awijan [OPTIONS] --end <END> <REPO>

Arguments:
  <REPO>  Source repository

Options:
      --target <TARGET>  Target branch [default: main]
      --end <END>        The PR number to stop at (last PR from previous release)
      --team <TEAM>      Team members who don't needed to be called out
  -h, --help             Print help
```
